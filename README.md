

## 运行 Rust

### 运行example

```bash
cargo run --example install
```

### 运行tests

```bash
cargo test
```

### 运行benchmarks

```bash
cargo bench
```
    
### 运行docs

```bash
cargo doc --no-deps
```

### 运行clippy

```bash
cargo clippy
```

### 运行fmt

```bash
cargo fmt
```

### 运行build

```bash
cargo build
```

### 运行check

```bash
cargo check
```

### 运行install

```bash
cargo install
